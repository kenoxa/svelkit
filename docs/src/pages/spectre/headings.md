# Headings

## heading(type)

```example
<script>
  import { heading } from '@svelkit/spectre'
</script>

<div use:heading={1}>heading(1)</div>
<div use:heading={2}>heading(2)</div>
<div use:heading={3}>heading(3)</div>
<div use:heading={4}>heading(4)</div>
<div use:heading={5}>heading(5)</div>
<div use:heading={6}>heading(6)</div>
```

## h\*

```example
<script>
  import { h1, h2, h3, h4, h5, h6 } from '@svelkit/spectre'
</script>

<div use:h1>h1</div>
<div use:h2>h2</div>
<div use:h3>h3</div>
<div use:h4>h4</div>
<div use:h5>h5</div>
<div use:h6>h6</div>
```
